# Desafio 3 – Criação de Grupos LearnIT

Este diretório contém as evidências da configuração e criação dos grupos organizacionais no Microsoft 365, conforme as diretrizes do projeto LearnIT.

##  Grupos Configurados

Para este desafio, foram criados grupos com diferentes finalidades e tipos de acesso:

* **GRP-Marketing-LearnIT** (Tipo: Microsoft 365): Destinado à colaboração da equipa de marketing, permitindo o uso partilhado de ficheiros, calendário e integração com o Microsoft Teams.
* **SG-Finance-LearnIT** (Tipo: Segurança): Criado especificamente para a gestão de permissões e segurança de dados do departamento financeiro.
