# Desafio 5 – Licenciamento por Grupo LearnIT

Nesta etapa, implementei o modelo de **licenciamento escalável** da LearnIT, garantindo que a atribuição de recursos seja feita de forma automática e eficiente.

##  Implementação

* **Grupo de Segurança**: Criado o grupo `LIC-M365-Business-LearnIT` para centralizar a gestão de subscrições.
* **Automação de Licenças**: Atribuímos a licença diretamente ao grupo, permitindo que qualquer membro adicionado receba os serviços sem intervenção manual.
