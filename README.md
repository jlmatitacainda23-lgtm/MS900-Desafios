# Projeto de Administração Microsoft 365 – LearnIT

Este repositório documenta a implementação de uma infraestrutura de colaboração e produtividade baseada no ecossistema Microsoft 365 para a organização fictícia **LearnIT**.

##  Objetivos do Projeto
O objetivo principal foi configurar e gerir um ambiente organizacional seguro e escalável, cobrindo desde a gestão de identidades até o ciclo de vida dos colaboradores.

##  Desafios Implementados

* **Gestão de Identidades**: Criação e configuração de utilizadores ativos conforme a estrutura organizacional.
* **Estrutura de Grupos**: Implementação de Grupos de Segurança e Listas de Distribuição para os departamentos de Marketing e Finanças.
* **Colaboração e Acesso**: Atribuição estratégica de membros a grupos para facilitar a partilha de recursos.
* **Licenciamento Escalável**: Configuração de licenciamento baseado em grupos de segurança, otimizando a gestão de subscrições.
* **Gestão de Ciclo de Vida (Offboarding)**: Execução de protocolos de segurança para suspensão de acessos e eliminação de utilizadores com foco em retenção de dados.

##  Estrutura do Repositório
O projeto está dividido em pastas que representam cada etapa crítica da configuração:
* `/desafio-01/`: Criação de Utilizadores.
* `/desafio-03/`: Criação de Grupos.
* `/desafio-04/`: Gestão de Membros.
* `/desafio-05/`: Licenciamento por Grupo.
* `/desafio-06/`: Gestão do Ciclo de Vida (Bloqueio e Eliminação).

##  Tecnologias Utilizadas
* Centro de Administração do Microsoft 365.
